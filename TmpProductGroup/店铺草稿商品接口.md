##1.  店铺临时商品接口

***
#### 1. 店铺中所有待审核商品
######  URL
/TmpProductGroup/AllProductGroup/
###### HTTP请求方式
GET
###### 请求参数
```
{
	'objectId': '12312341234',
	'page_nums': 'xx',
	'page': 'xx',
}
```
###### 返回数据
```
{
    'is_audit': 1,
    'paginator':{
        'has_next': has_next,
        'next_page_number': next_page_number,
        'number': page,
        'has_previous': has_previous,
        'previous_page_number': previous_page_number,
        'num_pages': num_pages
    },
    'productGroup': [
        {
            'objectId': objectId,
            'mainImage': {
                'url': url,
            },
            'briefDescription': briefDescription,
            'name': name,
            'is_shopProductGroup': is_shopProductGroup,
            'uniqueId': uniqueId,
        }
    ],
    'shop': {
        'uniqueId': uniqueId,
        'address': address,
        'name': name,
        'phoneNumber': phoneNumber,
        'objectId': objectId
    }

}
```


#### 2. 店铺创建商品
######  URL
/TmpProductGroup/CreateProductGroup/
###### HTTP请求方式
POST，GET
###### 请求参数
```
{
	'_shopObjectId': 'xxxxx'
}
or
{
    'productGroupObjectId': 'xxx',
    'delete': true,
}
```
###### 返回参数
```
{
    'content': {
        'data': {
            'shopObjectId': 'xxx',
            'saleCategory': [
                {
                    'objectId': 'objectId',
                    'name': 'name',
                    'value': [
                        {
                            'objectId': 'objectId',
                            'name': 'name',
                        }
                    ],
                }
            ],
            'storeCategory: [
                {
                    'objectId': 'objectId',
                    'name': 'name',
                    'value': [
                        {
                            'objectId': 'objectId',
                            'name':  'name',
                            'value': [
                                {
                                    'objectId': 'objectId',
                                    'name': 'name',
                                }
                            ]
                        }
                    ]
                }
            ],
            'productService: [
                {
                    'objectId': objectId,
                    'name': name,
                }
            ]
        }
    }
}

```

}

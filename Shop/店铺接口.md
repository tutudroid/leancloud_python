## 1.  店铺相关接口

***

#### 1. 店铺详情
######  URL
/Shop/ShopDetail/
###### HTTP请求方式
POST，GET
###### 请求参数
```
{
    objectId: xxx,
}
```
###### 返回参数
```
{
    'uniqueId': uniqueId,
    'user: userId,
    'address': address,
    'PROVINCE': province,
    'CITY': city,
    'DISTRICT': district,
    'name': name,
    'state': state,
    'createdAt': createdAt,
    'objectId': objectId,
    'phoneNumber': phoneNumber,
    'mainImage': mainImage,
}
```

```
}
```
